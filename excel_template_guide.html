‚öôÔ∏è Default Values

When importing, the following values will be automatically set:

  • Source: "Data Sheet"
  • Status: "Follow Up"
  • Budget: 0
  • Assigned To: Current user
  • Created By: Current user

üí° Important Notes:

  • Only rows with a phone number will be imported
  • Duplicate phone numbers will be skipped
  • The first sheet of your Excel file will be used
  • Supported formats: .xlsx and .xls
  • Column names are case-insensitive

üöÄ How to Import

  1. Prepare your Excel file according to the format above
  2. Go to the Leads Management page
  3. Click the "Import Excel" button (green button with upload icon)
  4. Select your Excel file
  5. Wait for the import to complete
  6. Check the success/error messages

‚ùì Troubleshooting

If import fails:

  • Make sure your Excel file has the correct column headers
  • Ensure phone numbers are in the correct format
  • Check that the file is not corrupted
  • Try saving the file as .xlsx format
  • Make sure the file is not too large (recommended: less than 1000 rows)
